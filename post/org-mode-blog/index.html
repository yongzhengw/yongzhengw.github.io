1. 安装 Hugo

参考: Quick Start

1.1 切换主题

  1. clone 主题到 themes 下
  2. 修改 `config.toml`, `theme = “ananke”` 即可

参考: Step 3: Add a Theme

git submodule add https://github.com/olOwOlo/hugo-theme-even themes/even

2. 安装 ox-hugo

参考: Installation

安装完成后，执行 C-c C-e 没看到 ox-hugo 选项，需在配置中修改 dotspacemacs-additional-packages '(ox-hugo)
参考: Issue with having ox-hugo installed on Spacemacs

3. ox-hugo 用法

front-matter

#+title: 2017-07-24
#+date: 2017-07-24
#+hugo_publishdate: 2018-01-26
#+hugo_expirydate: 2999-01-01
#+hugo_auto_set_lastmod: t
#+hugo_tags: toto zulu
#+hugo_categories: x y
#+hugo_draft: true
#+hugo_draft: false
#+hugo_weight: 123
#+hugo_weight: :tags 123
#+hugo_weight: :categories 123
#+hugo_menu: :menu foo :weight 123

参考: Doc

YASnippet

#+options: author:nil
#+OPTIONS: d:nil
#+EXCLUDE_TAGS: noexport
#+HUGO_BASE_DIR: /Users/jim/Dropbox/yongzhengw.github.io
#+HUGO_AUTO_SET_LASTMOD: t
#+HUGO_DRAFT: false
#+title: $0

调 bug

M-x org-hugo-debug-info 能看到基本的信息

修改默认导出路径

参考: 修改 ox-hugo 默认 export path, 修改 emacs 下的 org-hugo-default-section-directory 变量即可
